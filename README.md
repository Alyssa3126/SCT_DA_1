# SCT_DA_1
Interactive Sales Performance Dashboard for SuperStore Data using Excel, Pivot Tables and Power Query.
# Project Overview
This project involves transforming raw Superstore sales data into an interactive, insight-driven dashboard using Microsoft Excel. The goal was to identify key performance indicators (KPIs) and sales trends to assist in business decision-making.
​# key Technical Steps
​Data Cleaning: Leveraged Power Query Editor to handle missing values, standardize data formats (specifically dates and currency), and remove duplicates to ensure data integrity.
​Data Analysis: Created multiple Pivot Tables to aggregate sales by category, region, shipping mode, and segment.
​Visualization: Designed a dynamic dashboard featuring Bar Charts, Line Graphs, and Doughnut Charts.
​Applied Conditional Formatting (Data Bars and Color Scales) to Pivot Tables to instantly highlight top-performing categories and declining trends.
​Interactivity: Integrated Slicers to allow users to filter the entire dashboard by region, making the report fully interactive.
​# Insights Discovered
​Top Performer: Technology consistently leads in sales and profit compared to other categories.
​Trend Analysis: Sales experienced a significant peak in 2018, as visualized in the Year-Wise trend line.
​Regional Distribution: The West region contributes the largest share of total revenue
